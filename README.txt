============================================
FLavor Fusion Restaurant Website
============================================

HOW TO EDIT THE WEBSITE:
-----------------------
1. EDIT RESTAURANT INFO:
   - Open: config/restaurant.txt
   - Change values after = sign
   - Save the file

2. EDIT MENU:
   - Open: config/menu.txt
   - Format: category|name|price|description|tags
   - Example: veg|Paneer Tikka|260|Marinated paneer|popular,chef
   - Save the file

3. ADD GALLERY IMAGES:
   - Upload images to /images folder
   - Open: config/gallery.txt
   - Add: title|description|filename.jpg
   - Save the file

4. ADD BRANCHES:
   - Open: config/branches.txt
   - Add: name|phone|address|timings|manager
   - Save the file

CHANGES WILL APPEAR INSTANTLY ON WEBSITE!
No coding required. Just edit text files.

============================================
NETLIFY DEPLOYMENT INSTRUCTIONS:
============================================

1. Create GitHub repository
2. Push all files to GitHub
3. Go to Netlify.com
4. Click "New site from Git"
5. Select your repository
6. Click "Deploy site"

Your website will be live at: your-site.netlify.app
